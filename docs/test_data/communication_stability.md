# 通信稳定性测试原始数据

## 测试说明

- **测试方法**：Linux侧连续发送1000帧数据，统计RT-Thread成功接收并正确响应的帧数
- **延迟测量**：Linux发送时记录时间戳T1，RT-Thread收到后立即回传，Linux收到回传时记录T2，单程延迟 = (T2 - T1)/2
